## やることリスト（時系列順）

### STEP 1: 事前準備

- [ ] **個人の実印を用意する**
  - 未登録の場合は八千代市役所で印鑑登録をする
- [ ] **印鑑証明書を取得する**（発行から3ヶ月以内のもの）
  - 八千代市役所 or マイナンバーカードでコンビニ交付
  - **必要枚数: 1通**（オンライン申請の場合は原本郵送が必要）
- [ ] **会社の実印（代表者印）を作成する**
  - 丸印（直径18mm以内）を印鑑屋で注文
  - 彫刻内容：「合同会社MUD-Ai 代表社員之印」
  - 銀行印・角印もセットで作ると便利
  - ネット注文なら最短翌日届くところもある

### STEP 2: マネーフォワードで会社情報を入力

- [ ] **マネーフォワード クラウド会社設立にアカウント登録**
- [ ] **会社情報を入力する**（MFの画面に従って以下を入力）
  - 会社形態: 合同会社
  - 商号: 合同会社MUD-Ai
  - 本店所在地: 千葉県八千代市緑が丘西1-4-4-909
  - 代表社員: 山川 健太
  - 資本金: 100,000円
  - 事業年度: 1月1日〜12月31日
  - 公告の方法: 官報に掲載
  - 事業目的: → `01_定款.md` の第2条を参照して入力
- [ ] **電子定款の作成を依頼する**
  - MF提携の行政書士が電子署名してくれる（約5,000円）
  - 依頼後、通常1〜3営業日で完成
  - ※合同会社は公証役場での認証不要なので、電子署名のみでOK

### STEP 3: 資本金の払い込み

- [ ] **電子定款の作成日以降に、資本金10万円を振り込む**
  - 自分名義の個人口座に振り込む
  - 振込名義が「ヤマカワ ケンタ」と表示されるように
  - 別の自分の口座からATM振込 or ネットバンキングで振込
- [ ] **払込の証拠を用意する**
  - ネットバンキング: 取引明細のスクリーンショットを保存
  - 通帳: 表紙・裏表紙・振込記載ページの3点をコピー
  - MFの画面で画像アップロードする場合もあり

### STEP 4: 登記書類の出力・押印

- [ ] **MFから登記書類一式をダウンロード・印刷する**
  - MFが自動生成してくれる書類:
    - 設立登記申請書
    - 代表社員、本店所在地及び資本金決定書
    - 代表社員の就任承諾書
    - 払込証明書の表紙
  - 各書類の内容は `01`〜`06` のファイルで事前確認できます
- [ ] **印刷した書類に押印する**
  - 会社代表者印（法人実印）と個人実印を使い分ける
  - どの書類にどの印鑑を押すかはMFの案内に従う
- [ ] **払込証明書を綴じる**
  - MF出力の表紙 + 通帳コピー（or 明細）をホチキス留め
  - 見開き部分に会社代表者印で契印

### STEP 5: オンライン登記申請

- [ ] **マイナンバーカードを用意する**（電子署名に必要）
  - ICカードリーダー or スマホ（JPKI対応アプリ）を使用
- [ ] **MFの案内に従い、登記・供託オンライン申請システムで申請**
  - MFが申請データの作成をサポートしてくれる
  - 電子署名はマイナンバーカードで行う
- [ ] **登録免許税を納付する**
  - **60,000円** をオンライン（電子納付 / ペイジー）で支払い
  - ※窓口申請の場合は収入印紙
- [ ] **添付書類の原本を法務局に郵送する**
  - 印鑑届出書（会社代表者印 + 個人実印を押印）
  - 印鑑証明書（個人・原本）
  - 払込証明書（表紙 + 通帳コピー綴じたもの）
  - 宛先: 千葉地方法務局 佐倉支局
  - **書留郵便** で送付
- [ ] **申請日を確認する**
  - **申請日 ＝ 会社設立日** なので日付に注意

### STEP 6: 登記完了後の手続き

- [ ] **登記完了を確認する**（申請から約1〜2週間）
  - 法務局から補正の連絡がなければ完了
- [ ] **登記事項証明書（登記簿謄本）を取得する**
  - オンライン請求（登記ねっと）で1通500円 → 郵送受取
  - 法務局窓口で1通600円
  - **3〜5通取得推奨**（銀行口座開設・税務届出等で必要）
- [ ] **印鑑カードを取得する**
  - 千葉地方法務局 佐倉支局に「印鑑カード交付申請書」を提出
  - ※郵送でも可能
- [ ] **法人の銀行口座を開設する**
  - 登記簿謄本・法人印鑑証明書・代表者の本人確認書類が必要
  - マネーフォワードからそのまま法人口座開設も案内あり

### STEP 7: 設立後の届出（MFで一部自動作成可能）

- [ ] **税務署に届出を提出する**
  - 法人設立届出書（設立日から2ヶ月以内）
  - 青色申告承認申請書（設立日から3ヶ月以内 ※必ず出す）
  - 給与支払事務所等の開設届出書
  - 源泉所得税の納期の特例承認申請書
  - → MFの「会社設立後の届出」機能でフォームを自動生成可能
  - → e-Taxで電子提出も可能
- [ ] **千葉県税事務所に届出**
  - 法人設立届出書
- [ ] **八千代市役所に届出**
  - 法人設立届出書
- [ ] **年金事務所に届出**（該当する場合）
  - 役員報酬を出す場合は社会保険加入が必要（設立日から5日以内）
  - 当面は役員報酬ゼロなら届出不要
