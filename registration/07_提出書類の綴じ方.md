# オンライン申請時の提出ガイド

> マネーフォワード経由のオンライン登記申請における提出物の整理です。

---

## オンラインで提出するもの

MFの案内に従い、登記・供託オンライン申請システムから送信します。

| 書類 | 備考 |
|------|------|
| 設立登記申請書（データ） | MFが自動生成 |
| 電子定款 | MF提携行政書士が電子署名済み |
| 登記すべき事項（データ） | MFが自動生成 |

電子署名にはマイナンバーカード + ICカードリーダー（またはスマホ）が必要です。

---

## 法務局に郵送するもの

オンラインで送信できない原本書類を書留で郵送します。

### 郵送物チェックリスト

- [ ] 代表社員、本店所在地及び資本金決定書（個人実印押印済み）
- [ ] 代表社員の就任承諾書（個人実印押印済み）
- [ ] 払込証明書（表紙 + 通帳コピー、ホチキス綴じ、契印済み）
- [ ] 印鑑届出書（会社代表者印 + 個人実印押印済み）
- [ ] 印鑑証明書（個人・原本）

### 宛先

```text
〒285-8503
千葉県佐倉市弥勒町92番地
千葉地方法務局 佐倉支局
「登記申請書類在中」と朱書き
```

- 書留郵便で送付
- オンライン申請日から2週間以内に届くようにする

---

## 登録免許税の納付

- 金額: 60,000円
- オンライン申請の場合: 電子納付（ペイジー / インターネットバンキング）
- MFの画面から納付手続きが案内されます

---

## 登記完了後にやること

申請から約1〜2週間で登記完了。法務局から補正連絡がなければOK。

### すぐやること

- [ ] 登記事項証明書を取得（3〜5通）
  - オンライン請求（登記ねっと）1通500円 or 法務局窓口1通600円
- [ ] 印鑑カードを取得
  - 佐倉支局に「印鑑カード交付申請書」を提出（郵送可）
- [ ] 法人の銀行口座を開設
  - MFから法人口座開設の案内あり

### 税務届出（MFで一部フォーム自動生成可能）

| 届出先 | 届出書類 | 期限 |
|--------|----------|------|
| 税務署 | 法人設立届出書 | 設立日から2ヶ月以内 |
| 税務署 | 青色申告承認申請書 | 設立日から3ヶ月以内 |
| 税務署 | 給与支払事務所等の開設届出書 | 開設日から1ヶ月以内 |
| 税務署 | 源泉所得税の納期の特例承認申請書 | 随時（早めに） |
| 千葉県税事務所 | 法人設立届出書 | 各自治体規定による |
| 八千代市役所 | 法人設立届出書 | 各自治体規定による |
| 年金事務所 | 健康保険・厚生年金 新規適用届 | 設立日から5日以内 |

> 役員報酬を当面ゼロにする場合、年金事務所への届出は不要です。
